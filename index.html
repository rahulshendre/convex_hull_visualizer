Asymptotic growth (Big‑O intuition)

Compare how work grows with input size n using asymptotic growth (Big‑O / Big‑Θ style). Curves plot n log2 n, n2, and n3 only - not stopwatch time. Constants and lower terms are omitted. Each curve is normalized to height 1 so they fit one plot; the y axis is not seconds - only relative shape vs n.
